EliteDash v1.0.0

Thank you for purchasing EliteDash. This documentation will help you set up and customize the dashboard.

1. Project Structure

  • src/
    • App.jsx (Main Controller)
    • OrderTerminal.jsx (Assets & Table)
    • ProfileSecurity.jsx (Security Settings)
    • NotificationCenter.jsx (Alerts System)
    • index.css (Glassmorphism Styles)
  • package.json (Dependencies)
  • tailwind.config.js (UI Configuration)

2. Installation Guide

Follow these steps to get the project running on your local machine:

# 1. Extract the zip file
# 2. Open terminal in the project folder
# 3. Install dependencies
npm install

# 4. Start the development server
npm start

3. Customization

To change the primary colors, edit tailwind.config.js. For the glassmorphism intensity, modify the .glass-panel class in index.css.
