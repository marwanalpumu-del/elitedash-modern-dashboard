# 💎 EliteDash - Premium Glassmorphism React Dashboard

EliteDash is a high-end, modern dashboard template built with **React 18** and **Tailwind CSS**. Designed with a "Terminal" aesthetic, it features a stunning glassmorphism UI/UX.

### 🚀 Key Features:
- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop.
- **Glassmorphic UI:** Cutting-edge blur and transparency effects.
- **Modular Components:** 5+ Dedicated screens (Assets, Security, Notifications).
- **Clean Code:** Fully documented and easy to customize.
- **Icon Pack:** Integrated with Lucide-React.

### 🛠️ Installation:
1. Clone the repo.
2. Run `npm install`.
3. Run `npm start`.
