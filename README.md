# 🚀 EliteDash - Premium Glassmorphism Admin Dashboard

EliteDash is a modern, high-performance admin dashboard template built with **Tailwind CSS**, **Lucide Icons**, and **ApexCharts**. It features a stunning **Glassmorphism** design language inspired by Apple’s UI, offering a seamless user experience with real-time data persistence.

![Version](https://img.shields.io/badge/version-1.0.0-green.svg)
![TailwindCSS](https://img.shields.io/badge/Tailwind-CSS-38B2AC?logo=tailwind-css)

## ✨ Key Features

- **🌓 Dual-Theme System**: Instant switching between Dark and Light modes with persistent memory.
- **🖼️ Profile Management**: Real-time profile picture upload (Base64) and name synchronization across all pages.
- **📊 Interactive Analytics**: Beautiful, responsive charts powered by ApexCharts that adapt to theme changes.
- **💎 Premium UI**: Ultra-modern Glassmorphism cards, smooth transitions, and refined typography.
- **📱 Fully Responsive**: Optimized for desktops, tablets, and mobile devices.
- **💾 Local Persistence**: Uses `LocalStorage` to save user preferences, theme, and profile data without a backend.
- **⚠️ Custom 404 Page**: A branded, stylish error page to keep the user experience consistent.

## 🛠️ Tech Stack

- **Framework**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Charts**: [ApexCharts](https://apexcharts.com/)
- **Fonts**: Inter (Google Fonts)
- **Logic**: Vanilla JavaScript (ES6+)
