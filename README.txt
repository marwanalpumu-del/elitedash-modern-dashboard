------------------------------------------------------------
📂 PROJECT STRUCTURE
------------------------------------------------------------
- /src           : Core React components and application logic.
- /public        : Static assets and index.html.
- /Documentation : Detailed HTML guide for setup and customization.
- package.json   : Project dependencies and scripts.
- tailwind.config.js : Theme and styling configurations.

------------------------------------------------------------
🚀 QUICK START GUIDE
------------------------------------------------------------
To get the project up and running, follow these steps:

1. Extract the "EliteDash_Project" folder.
2. Open your terminal/command prompt in the extracted directory.
3. Install dependencies by running:
   npm install

4. Launch the development server by running:
   npm start

5. Open http://localhost:3000 in your browser to view the terminal.
